# Fsync
[![](https://img.shields.io/badge/version-0.0.1-blue.svg)]()

Fsync是用于实现不同机器上的数据实时同步功能。对于会同时发生变化的文件，服务器端会观察文件的变化，并同时生成变化文件的校验和，客户端会定时地将本地的校验和与服务器端的校验和进行比较，对于发生变化的文件发起同步请求。

## 特点
1. 对文件进行压缩进行同步
2. 内部有哈希树维护各个文件的哈希
3. 文件同步完成之后还会执行reload操作
4. 基于inotify发现词典变化
5. 托管应用的启动
6. 保留多个版本进行备份
7. 异步下载和解压、压缩文件
